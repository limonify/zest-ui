`Collapsible` is one open/close disclosure. `Collapsible.Panel` publishes its measured height and `transitionStatus`; keep it mounted with `keepMounted` to animate the exit.

## Usage

The behaviour is headless — the **Code** tab is the component; the **StyleSheet** and **Uniwind** tabs are two ways to design it.

<Tabs items={['Code', 'StyleSheet', 'Uniwind']}>

<Tab value="Code">

```tsx
import { Text } from 'react-native';
import { Collapsible } from '@limonify/zest-ui';

<Collapsible.Root defaultOpen>
  <Collapsible.Trigger style={styles.trigger}>
    <Text style={styles.triggerText}>Toggle details</Text>
  </Collapsible.Trigger>
  {/* Rendered only while open, so it lays out at its natural height. */}
  <Collapsible.Panel>
    <Text style={styles.body}>The panel content.</Text>
  </Collapsible.Panel>
</Collapsible.Root>
```

## Animation

`Collapsible.Panel` publishes its measured content height on `state.height` plus `state.transitionStatus`. Drive a shared value from `state.open` and clip to the measured height:

```tsx
import Animated, { useSharedValue, useAnimatedStyle, withTiming } from 'react-native-reanimated';
import { useEffect } from 'react';
import { Collapsible } from '@limonify/zest-ui';

function Panel({ props, state }) {
  const progress = useSharedValue(state.open ? 1 : 0);

  useEffect(() => {
    progress.value = withTiming(state.open ? 1 : 0, { duration: 200 });
  }, [state.open]);

  const animatedStyle = useAnimatedStyle(() => ({
    height: state.height != null ? progress.value * state.height : undefined,
    opacity: progress.value,
  }));

  return <Animated.View {...props} style={[{ overflow: 'hidden' }, animatedStyle]} />;
}

<Collapsible.Panel keepMounted render={(props, state) => <Panel props={props} state={state} />}>
  {/* ... */}
</Collapsible.Panel>
```

See the [Reanimated guide](/docs/guides/animation-reanimated) for the full contract.

## Accessibility

The trigger carries `aria-expanded`; the panel is hidden from assistive tech while closed.
