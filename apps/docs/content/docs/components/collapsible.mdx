`Collapsible` is one open/close disclosure. `Collapsible.Panel` publishes its measured height and `transitionStatus`; keep it mounted with `keepMounted` to animate the exit.

## Usage

The behaviour is headless â€” the **Code** tab is the component; the **StyleSheet** and **Uniwind** tabs are two ways to design it.

<Tabs items={['Code', 'StyleSheet', 'Uniwind']}>

<Tab value="Code">

```tsx
import { Text } from 'react-native';
import { Collapsible } from '@limonify/zest';

<Collapsible.Root defaultOpen>
  <Collapsible.Trigger style={styles.trigger}>
    <Text style={styles.triggerText}>Toggle details</Text>
  </Collapsible.Trigger>
  {/* Rendered only while open, so it lays out at its natural height. */}
  <Collapsible.Panel>
    <Text style={styles.body}>The panel content.</Text>
  </Collapsible.Panel>
</Collapsible.Root>
```

</Tab>

<Tab value="StyleSheet">

```tsx
import { StyleSheet } from 'react-native';

const styles = StyleSheet.create({
  trigger: { backgroundColor: '#171717', borderRadius: 10, padding: 12, alignSelf: 'flex-start' },
  triggerText: { color: '#fff', fontWeight: '600' },
  body: { paddingVertical: 12, color: '#444' },
});
```

</Tab>

<Tab value="Uniwind">

```tsx

<Collapsible.Root defaultOpen>

    <Text className="font-semibold text-white">Toggle details</Text>
  </Collapsible.Trigger>
  <Collapsible.Panel>
    <Text className="py-3 text-neutral-600">The panel content.</Text>
  </Collapsible.Panel>

```

</Tab>

## Accessibility

The trigger carries `aria-expanded`; the panel is hidden from assistive tech while closed. See the [Reanimated guide](/docs/guides/animation-reanimated) for the height animation.
