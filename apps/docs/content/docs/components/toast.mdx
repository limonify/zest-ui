`Toast` is the one popup that is **not** a Modal — the app underneath must stay usable. Place `Toast.Provider` at the app root with a `Toast.Viewport`, and raise toasts imperatively with `useToastManager()`.

<Demo>A toast sliding in, pausing on press, and swiping away.</Demo>

## Usage

```tsx
import { Toast } from '@limonify/zest';

// At the app root:
<Toast.Provider timeout={4000} limit={3}>
  <App />
  <Toast.Viewport style={styles.viewport}>
    {/* map useToastManager().toasts to <Toast.Root> */}
  </Toast.Viewport>
</Toast.Provider>

// Anywhere inside:
const { add } = Toast.useToastManager();
add({ title: 'Saved', description: 'Your changes are safe.' });
```

## Accessibility

The viewport uses `pointerEvents="box-none"` so touches pass through to the app; only the toasts are interactive. `accessibilityLiveRegion="polite"` announces new toasts.
