`Tabs` switches between panels. `Tabs.Indicator` publishes the active tab's measured position and size on state, so you can slide an underline.

## Usage

The behaviour is headless — the **Code** tab is the component; the **StyleSheet** and **Uniwind** tabs are two ways to design it.

<Tabs items={['Code', 'StyleSheet', 'Uniwind']}>

<Tab value="Code">

```tsx
import { Animated, Text } from 'react-native';
import { Tabs } from '@limonify/zest-ui';

<Tabs.Root defaultValue="overview">
  <Tabs.List style={styles.list}>
    {tabs.map((t) => (
      <Tabs.Tab key={t.value} value={t.value} style={styles.tab}>
        <Text>{t.label}</Text>
      </Tabs.Tab>
    ))}
    <Tabs.Indicator
      render={(props, s) => (
        <Animated.View
          {...props}
          style={[styles.indicator, { left: s.selectedTabPosition?.left, width: s.selectedTabSize?.width }]}
        />
      )}
    />
  </Tabs.List>
  {tabs.map((t) => (
    <Tabs.Panel key={t.value} value={t.value}>
      <Text style={styles.body}>{t.body}</Text>
    </Tabs.Panel>
  ))}
</Tabs.Root>
```

</Tab>

## Animation

`Tabs.Indicator` publishes the active tab's `selectedTabPosition` (`{ left, right, top, bottom }`) and `selectedTabSize` (`{ width, height }`). Drive shared values from them so the bar slides between tabs instead of jumping:

```tsx
import Animated, { useSharedValue, useAnimatedStyle, withTiming } from 'react-native-reanimated';
import { useEffect } from 'react';
import { Tabs } from '@limonify/zest-ui';

function Indicator({ props, state }) {
  const left = useSharedValue(0);
  const width = useSharedValue(0);

  useEffect(() => {
    if (state.selectedTabPosition && state.selectedTabSize) {
      left.value = withTiming(state.selectedTabPosition.left, { duration: 180 });
      width.value = withTiming(state.selectedTabSize.width, { duration: 180 });
    }
  }, [state.selectedTabPosition, state.selectedTabSize]);

  const animatedStyle = useAnimatedStyle(() => ({ left: left.value, width: width.value }));

  return <Animated.View {...props} style={[styles.indicator, animatedStyle]} />;
}

<Tabs.Indicator render={(props, state) => <Indicator props={props} state={state} />} />
```

See the [Reanimated guide](/docs/guides/animation-reanimated) for the full contract.

## Accessibility

Tabs are `accessibilityRole="tab"`, panels `role="tabpanel"`. Item order is read from measured layout via `CompositeList`, so the indicator tracks the real visual order.
