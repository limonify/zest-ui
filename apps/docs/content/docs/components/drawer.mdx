---
title: Drawer
description: A dialog that also opens and dismisses by swiping, with snap points.
---

`Drawer` is a `Dialog` you can swipe away. `Drawer.Popup` publishes `swipeMovement` on state so you drive the transform; `Drawer.SwipeArea` opens it from a screen edge. Optional `snapPoints` make it a bottom sheet. Needs `react-native-gesture-handler`.

## Usage

The behaviour is headless — the **Code** tab is the component; the **StyleSheet** and **Uniwind** tabs are two ways to design it.

<Tabs items={['Code', 'StyleSheet', 'Uniwind']}>

<Tab value="Code">

```tsx
import { Text } from 'react-native';
import { Drawer } from '@limonify/zest';

<Drawer.Root swipeDirection="down">
  <Drawer.Trigger style={styles.button}><Text style={styles.buttonText}>Open</Text></Drawer.Trigger>
  <Drawer.Portal>
    <Drawer.Backdrop style={styles.backdrop} />
    <Drawer.Viewport style={styles.viewport}>
      <Drawer.Popup style={(s) => [styles.sheet, { transform: [{ translateY: s.swipeMovement }] }]}>
        <Drawer.Title style={styles.title}>Swipe me down</Drawer.Title>
        <Drawer.Close style={styles.button}><Text style={styles.buttonText}>Close</Text></Drawer.Close>
      </Drawer.Popup>
    </Drawer.Viewport>
  </Drawer.Portal>
</Drawer.Root>
```

## Animation

`Drawer.Popup` publishes `state.transitionStatus` and, while dragging, `state.swipeMovement`. The swipe-to-dismiss gesture is already built in; for the open/close use a slide preset matching the drawer's side, and fade the backdrop:

```tsx
import Animated, { SlideInDown, SlideOutDown, FadeIn, FadeOut } from 'react-native-reanimated';
import { Drawer } from '@limonify/zest';

<Drawer.Backdrop keepMounted render={(props) => (
  <Animated.View {...props} entering={FadeIn} exiting={FadeOut} />
)} />
<Drawer.Popup keepMounted render={(props) => (
  <Animated.View {...props} entering={SlideInDown} exiting={SlideOutDown} />
)}>
  {/* ... */}
</Drawer.Popup>
```

See the [Reanimated guide](/docs/guides/animation-reanimated) for the full contract.

## Accessibility

A bottom sheet is `<Drawer.Root snapPoints={[0.25, 0.5, 0.9]}>`. Swipe dismissal reports the `swipe` reason through `onOpenChange`.
