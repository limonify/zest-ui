---
title: Meter
description: A static gauge, like disk usage or battery level.
---

`Meter` is a static measurement, not a task in progress — so it has no indeterminate state and no `status`. Its parts mirror `Progress`.

<Demo>A storage-used gauge.</Demo>

## Usage

The behaviour is headless — the **Code** tab is the component; the **StyleSheet** and **Uniwind** tabs are two ways to design it.

<Tabs items={['Code', 'StyleSheet', 'Uniwind']}>

<Tab value="Code">

```tsx
import { Meter } from '@limonify/zest-ui';

<Meter.Root value={72}>
  <Meter.Label style={styles.label}>Storage used</Meter.Label>
  <Meter.Value style={styles.label} />
  <Meter.Track style={styles.track}>
    <Meter.Indicator style={styles.fill} />
  </Meter.Track>
</Meter.Root>
```

</Tab>

<Tab value="StyleSheet">

```tsx
import { StyleSheet } from 'react-native';

const styles = StyleSheet.create({
  label: { color: '#666', marginBottom: 6 },
  track: { height: 8, borderRadius: 4, backgroundColor: '#e5e5e5', overflow: 'hidden' },
  fill: { height: 8, backgroundColor: '#4c7a0b' },
});
```

</Tab>

<Tab value="Uniwind">

```tsx
import { Meter } from '@limonify/zest-ui';

<Meter.Root value={72}>
  <Meter.Label className="mb-1.5 text-neutral-500">Storage used</Meter.Label>
  <Meter.Value className="text-neutral-500" />
  <Meter.Track className="h-2 overflow-hidden rounded-full bg-neutral-200">
    <Meter.Indicator className="h-2 bg-lime-700" />
  </Meter.Track>
</Meter.Root>
```

</Tab>

</Tabs>

## Accessibility

`role="meter"` (with a `progressbar` accessibilityRole fallback) and `accessibilityValue`. The value normalizes to a percentage of `min`/`max` for the indicator width.
