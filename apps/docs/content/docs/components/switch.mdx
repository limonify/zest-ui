## Usage

The behaviour is headless — the **Code** tab is the component; the **StyleSheet** and **Uniwind** tabs are two ways to design it.

<Tabs items={['Code', 'StyleSheet', 'Uniwind']}>

<Tab value="Code">

```tsx
import { Switch } from '@limonify/zest-ui';

<Switch.Root
  checked={on}
  onCheckedChange={setOn}
  style={(state) => [styles.track, state.checked && styles.trackOn]}
>
  <Switch.Thumb style={(state) => [styles.thumb, state.checked && styles.thumbOn]} />
</Switch.Root>
```

</Tab>

<Tab value="StyleSheet">

import { StyleSheet } from 'react-native';

## Animation

`Switch.Thumb` state exposes `state.checked`. Slide the thumb with a shared value — the travel distance is your track width minus the thumb width:

```tsx
import Animated, { useAnimatedStyle } from 'react-native-reanimated';
import { useAnimatedValue } from './useAnimatedValue'; // small helper — see the guide
import { Switch } from '@limonify/zest-ui';

const TRAVEL = 20; // track width − thumb width

function Thumb({ props, state }) {
  const offset = useAnimatedValue(state.checked ? TRAVEL : 0, { duration: 160 });
  const style = useAnimatedStyle(() => ({ transform: [{ translateX: offset.value }] }));
  return <Animated.View {...props} style={[styles.thumb, style]} />;
}

<Switch.Root checked={on} onCheckedChange={setOn} style={styles.track}>
  <Switch.Thumb render={(props, state) => <Thumb props={props} state={state} />} />
</Switch.Root>
```

See the [Reanimated guide](/docs/guides/animation-reanimated) for the full contract.

## Accessibility

`accessibilityRole="switch"` with `accessibilityState.checked`. Field-aware inside a `Field.Root`.
