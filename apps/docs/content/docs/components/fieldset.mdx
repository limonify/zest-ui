`Fieldset` groups related fields under a legend; a disabled fieldset disables every field inside it.

<Demo>A disabled fieldset.</Demo>

## Anatomy

<Anatomy code={`Fieldset.Root
  Fieldset.Legend
  Field.Root …`} />

## Usage

```tsx
import { Fieldset, Field } from '@limonify/zest';

<Fieldset.Root disabled>
  <Fieldset.Legend>Shipping address</Fieldset.Legend>
  <Field.Root>{/* … */}</Field.Root>
</Fieldset.Root>
```

## Accessibility

`role="group"` labelled by `Fieldset.Legend`. A disabled fieldset disables every `Field.Root` inside it.
