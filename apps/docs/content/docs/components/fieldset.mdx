`Fieldset` groups related fields under a legend; a disabled fieldset disables every field inside it.

<Demo>A disabled fieldset.</Demo>

## Anatomy

<Anatomy code={`Fieldset.Root
  Fieldset.Legend
  Field.Root …`} />

## Usage

The behaviour is headless — the **Code** tab is the component; the **StyleSheet** and **Uniwind** tabs are two ways to design it.

<Tabs items={['Code', 'StyleSheet', 'Uniwind']}>

<Tab value="Code">

```tsx
import { Text } from 'react-native';
import { Fieldset, Field } from '@limonify/zest-ui';

<Fieldset.Root disabled style={styles.set}>
  <Fieldset.Legend style={styles.legend}>Shipping address</Fieldset.Legend>
  <Field.Root>
    <Field.Control style={styles.input} placeholder="Street" />
  </Field.Root>
</Fieldset.Root>
```

</Tab>

<Tab value="StyleSheet">

```tsx
import { StyleSheet } from 'react-native';

const styles = StyleSheet.create({
  set: { gap: 8 },
  legend: { fontWeight: '600' },
  input: { borderWidth: 1, borderColor: '#d4d4d4', borderRadius: 10, padding: 10 },
});
```

</Tab>

<Tab value="Uniwind">

```tsx
import { Fieldset, Field } from '@limonify/zest-ui';

<Fieldset.Root disabled className="gap-2">
  <Fieldset.Legend className="font-semibold">Shipping address</Fieldset.Legend>
  <Field.Root>
    <Field.Control className="rounded-lg border border-neutral-300 p-2.5" placeholder="Street" />
  </Field.Root>
</Fieldset.Root>
```

</Tab>

</Tabs>

## Accessibility

`role="group"` labelled by `Fieldset.Legend`. A disabled fieldset disables every `Field.Root` inside it.
