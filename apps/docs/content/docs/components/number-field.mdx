`NumberField` is a numeric text input with increment/decrement buttons and a drag-to-scrub area, with locale-aware parsing and formatting.

<Demo>Stepping and scrubbing a number.</Demo>

<Anatomy code={`NumberField.Root
  NumberField.Group

    NumberField.Increment
  NumberField.ScrubArea`} />

## Usage

```tsx
import { Text } from 'react-native';
import { NumberField } from '@limonify/zest';

<NumberField.Root defaultValue={5} min={0} max={10}>
  <NumberField.Group style={styles.group}>
    <NumberField.Decrement><Text>−</Text></NumberField.Decrement>
    <NumberField.Input style={styles.input} />
    <NumberField.Increment><Text>+</Text></NumberField.Increment>
  </NumberField.Group>
</NumberField.Root>
```

## Accessibility

The input is `accessibilityRole="spinbutton"` with `accessibilityValue`. Parsing/formatting is locale-aware via `format`/`locale`. Field-aware inside a `Field.Root`.
