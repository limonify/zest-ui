`RadioGroup` provides the selected value to a set of `Radio` buttons and commits changes before applying them, so a handler can veto.

<Demo>A radio group committing a value.</Demo>

## Anatomy

<Anatomy code={`RadioGroup
  Radio.Root`} />

## Usage

```tsx
import { RadioGroup, Radio } from '@limonify/zest';

<RadioGroup defaultValue="md" onValueChange={setSize}>
  {/* Radio.Root items */}
</RadioGroup>
```

`accessibilityRole="radiogroup"`. Inside a `Field.Root`, the group is labelled and validated by the field.
