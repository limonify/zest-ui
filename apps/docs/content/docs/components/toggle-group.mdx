`ToggleGroup` coordinates a set of toggles as a single- or multiple-selection control.

<Demo>A single-select formatting group.</Demo>

## Anatomy

<Anatomy code={`ToggleGroup
  Toggle`} />

## Usage

```tsx
import { ToggleGroup, Toggle } from '@limonify/zest';

<ToggleGroup value={value} onValueChange={setValue}>
  <Toggle value="left">{/* … */}</Toggle>
  <Toggle value="center">{/* … */}</Toggle>
</ToggleGroup>
```

Set `toggleMultiple` for multiple selection. The group has `role="group"`.
