`Slider` is a value (or range) selector driven by a drag gesture. It needs `react-native-gesture-handler` and the app wrapped in `GestureHandlerRootView`.

<Demo>Dragging a thumb, and a range slider.</Demo>

## Usage

```tsx
import { Slider } from '@limonify/zest';

<Slider.Root defaultValue={40}>
  <Slider.Value />
  <Slider.Control style={styles.control}>
    <Slider.Track style={styles.track}>
      <Slider.Indicator style={styles.fill} />
      <Slider.Thumb style={styles.thumb} />
    </Slider.Track>
  </Slider.Control>
</Slider.Root>
```

## Accessibility

Pass an array `value` for a range slider. Each `Slider.Thumb` is `accessibilityRole="adjustable"` and labelled by `Slider.Label`. Requires `react-native-gesture-handler`.
