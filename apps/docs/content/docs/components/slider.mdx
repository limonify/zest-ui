`Slider` is a value (or range) selector driven by a drag gesture. It needs `react-native-gesture-handler` and the app wrapped in `GestureHandlerRootView`.

## Animation

The built-in gesture already moves the thumb live, so you rarely animate its position. Instead react to `state.dragging` (on `Slider.Thumb`) — for example, grow the thumb while it is held:

```tsx
import Animated, { useAnimatedStyle } from 'react-native-reanimated';
import { useAnimatedValue } from './useAnimatedValue'; // small helper — see the guide
import { Slider } from '@limonify/zest-ui';

function Thumb({ props, state }) {
  const scale = useAnimatedValue(state.dragging ? 1.25 : 1, { duration: 120 });
  const style = useAnimatedStyle(() => ({ transform: [{ scale: scale.value }] }));
  return <Animated.View {...props} style={[styles.thumb, style]} />;
}

<Slider.Thumb render={(props, state) => <Thumb props={props} state={state} />} />
```

To make the thumb *follow* the finger with spring physics instead, feed `state.values` into a shared value — see **Gesture-following values** in the [Reanimated guide](/docs/guides/animation-reanimated).

## Accessibility

Pass an array `value` for a range slider. Each `Slider.Thumb` is `accessibilityRole="adjustable"` and labelled by `Slider.Label`. Requires `react-native-gesture-handler`.
