`Input` is a standalone `TextInput`. Used inside a `Field.Root` it picks up the field's label, description, error and validation automatically.

<Demo>A standalone input.</Demo>

## Anatomy

<Anatomy code={`Input`} />

## Usage

```tsx
import { Input } from '@limonify/zest';

<Input placeholder="Search" value={q} onValueChange={setQ} style={styles.input} />
```

## Accessibility

Used inside a `Field.Root`, `Input` is labelled and described by the field just like `Field.Control`.
