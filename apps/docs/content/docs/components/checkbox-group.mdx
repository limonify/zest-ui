`CheckboxGroup` shares one value array across several checkboxes, with an optional parent checkbox that reflects and toggles all its children.

<Demo>A parent checkbox toggling three children.</Demo>

## Anatomy

<Anatomy code={`CheckboxGroup
  Checkbox.Root (parent)
  Checkbox.Root (child, value=…)`} />

## Usage

The behaviour is headless — the **Code** tab is the component; the **StyleSheet** and **Uniwind** tabs are two ways to design it.

<Tabs items={['Code', 'StyleSheet', 'Uniwind']}>

<Tab value="Code">

```tsx
import { Text } from 'react-native';
import { CheckboxGroup, Checkbox } from '@limonify/zest-ui';

<CheckboxGroup value={value} onValueChange={setValue} allValues={COLORS}>
  <Checkbox.Root parent style={styles.box}>
    <Checkbox.Indicator><Text style={styles.tick}>—</Text></Checkbox.Indicator>
  </Checkbox.Root>
  {COLORS.map((c) => (
    <Checkbox.Root key={c} value={c} style={styles.box}>
      <Checkbox.Indicator><Text style={styles.tick}>✓</Text></Checkbox.Indicator>
    </Checkbox.Root>
  ))}
</CheckboxGroup>
```

</Tab>

<Tab value="StyleSheet">

```tsx
import { StyleSheet } from 'react-native';

const styles = StyleSheet.create({
  box: {
    width: 22, height: 22, borderRadius: 6, borderWidth: 1.5,
    borderColor: '#cbcbcb', alignItems: 'center', justifyContent: 'center',
  },
  tick: { color: '#4c7a0b', fontWeight: '700' },
});
```

</Tab>

<Tab value="Uniwind">

```tsx
import { Text } from 'react-native';
import { CheckboxGroup, Checkbox } from '@limonify/zest-ui';

<CheckboxGroup value={value} onValueChange={setValue} allValues={COLORS}>
  <Checkbox.Root parent className="size-6 items-center justify-center rounded-md border-[1.5px] border-neutral-300">
    <Checkbox.Indicator><Text className="font-bold text-lime-700">—</Text></Checkbox.Indicator>
  </Checkbox.Root>
  {/* children … */}
</CheckboxGroup>
```

</Tab>

</Tabs>

## Accessibility

The parent checkbox reports `'mixed'` when only some children are checked, and toggles all of them on press.
