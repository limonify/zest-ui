`CheckboxGroup` shares one value array across several checkboxes, with an optional parent checkbox that reflects and toggles all its children.

<Demo>A parent checkbox toggling three children.</Demo>

## Anatomy

<Anatomy code={`CheckboxGroup
  Checkbox.Root (parent)
  Checkbox.Root (child, value=…)`} />

## Usage

```tsx
import { CheckboxGroup, Checkbox } from '@limonify/zest';

<CheckboxGroup value={value} onValueChange={setValue} allValues={COLORS}>
  <Checkbox.Root parent>{/* indicator */}</Checkbox.Root>
  {COLORS.map((c) => (
    <Checkbox.Root key={c} value={c}>{/* indicator */}</Checkbox.Root>
  ))}
</CheckboxGroup>
```

## Accessibility

The parent checkbox reports `'mixed'` when only some children are checked, and toggles all of them on press.
