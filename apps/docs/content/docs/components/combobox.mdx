title: Combobox

---

`Combobox` is a text input paired with a filtered list. Typing filters the items; choosing one
fills the input and closes the list. For free-text suggestions without a bound value, use
[`Autocomplete`](/docs/components/combobox), which reuses every combobox part.

## Usage

The behaviour is headless — the **Code** tab is the component; the **StyleSheet** and **Uniwind** tabs are two ways to design it.

<Tabs items={['Code', 'StyleSheet', 'Uniwind']}>

<Tab value="Code">

```tsx
import { Text } from 'react-native';
import { Combobox } from '@limonify/zest';

<Combobox.Root items={COUNTRIES} value={value} onValueChange={setValue}>
  <Combobox.Input placeholder="Pick a country" style={styles.input} />
  <Combobox.Portal>
    <Combobox.Backdrop style={styles.transparent} />
    <Combobox.Positioner>
      <Combobox.Popup style={styles.popup}>
        <Combobox.Empty style={styles.empty}><Text>No match</Text></Combobox.Empty>
        <Combobox.List>
          {(item) => (
            <Combobox.Item key={String(item.value)} item={item} style={styles.item}>
              <Text>{item.label}</Text>
            </Combobox.Item>
          )}
        </Combobox.List>
      </Combobox.Popup>
    </Combobox.Positioner>
  </Combobox.Portal>
</Combobox.Root>
```

</Tab>

## API

`Combobox.Root`

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `items` | `(string \| { value, label })[]` | — | The items to filter and choose from. |
| `value` / `defaultValue` | `unknown` | — | The selected value. |
| `onValueChange` | `(value) => void` | — | Called when the selection changes. |
| `inputValue` / `defaultInputValue` | `string` | — | The input text. |
| `openOnFocus` | `boolean` | `true` | Open the list when the input is focused. |
| `filter` | `(item, query) => boolean` | label match | Custom filter predicate. |

## Accessibility

The input is `accessibilityRole="search"` with `aria-expanded` and `aria-autocomplete="list"`; each
item is `role="option"` with `aria-selected`.
