---
title: Accordion
description: A set of collapsible sections, one or many open at a time.
---

`Accordion` groups several collapsible items. Each `Accordion.Panel` publishes its measured height and `transitionStatus` on state so you drive the open/close animation.

## Usage

The behaviour is headless — the **Code** tab is the component; the **StyleSheet** and **Uniwind** tabs are two ways to design it.

import { Pressable, Text } from 'react-native';
import { Accordion } from '@limonify/zest';

<Accordion.Root defaultValue={['shipping']}>

    <Accordion.Item key={item.value} value={item.value} style={styles.item}>

## Animation

`Accordion.Panel` publishes its measured content height on `state.height` plus `state.transitionStatus`. Drive a shared value from `state.open` and clip to the measured height:

```tsx
import Animated, { useSharedValue, useAnimatedStyle, withTiming } from 'react-native-reanimated';
import { useEffect } from 'react';
import { Accordion } from '@limonify/zest';

function Panel({ props, state }) {
  const progress = useSharedValue(state.open ? 1 : 0);

  useEffect(() => {
    progress.value = withTiming(state.open ? 1 : 0, { duration: 200 });
  }, [state.open]);

  const animatedStyle = useAnimatedStyle(() => ({
    height: state.height != null ? progress.value * state.height : undefined,
    opacity: progress.value,
  }));

  return <Animated.View {...props} style={[{ overflow: 'hidden' }, animatedStyle]} />;
}

<Accordion.Panel keepMounted render={(props, state) => <Panel props={props} state={state} />}>
  {/* ... */}
</Accordion.Panel>
```

See the [Reanimated guide](/docs/guides/animation-reanimated) for the full contract.

## Accessibility

Set `openMultiple` to keep several open. See the [Reanimated guide](/docs/guides/animation-reanimated) for animating the panel; render only while open for a robust natural-height layout.
