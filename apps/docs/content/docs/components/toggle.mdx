`Toggle` is a two-state pressable button (pressed / not pressed) — a bold button, a formatting toggle.

<Demo>Pressing a bold toggle.</Demo>

## Anatomy

<Anatomy code={`Toggle`} />

## Usage

```tsx
import { Text } from 'react-native';
import { Toggle } from '@limonify/zest';

<Toggle
  pressed={bold}
  onPressedChange={setBold}
  style={(state) => [styles.toggle, state.pressed && styles.togglePressed]}
>
  <Text>B</Text>
</Toggle>
```

## Accessibility

`accessibilityState.selected` reflects the pressed state; the toggle works standalone or inside a `ToggleGroup`.
