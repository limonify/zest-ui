title: Button
description: A pressable that tracks its own pressed and disabled state.
---

`Button` renders a `Pressable` with `accessibilityRole="button"` and exposes `pressed`/`disabled`
on its state, so you can style each visually without tracking press state yourself.

## Usage

The behaviour is headless â€” the **Code** tab is the component; the **StyleSheet** and **Uniwind** tabs are two ways to design it.

<Tabs items={['Code', 'StyleSheet', 'Uniwind']}>

<Tab value="Code">

```tsx
import { Text } from 'react-native';
import { Button } from '@limonify/zest-ui';

<Button
  onPress={() => save()}
  style={(state) => [
    styles.button,
    state.pressed && styles.buttonPressed,
    state.disabled && styles.buttonDisabled,
  ]}
>
  <Text style={styles.text}>Save</Text>
</Button>
```

</Tab>

<Tab value="StyleSheet">

```tsx
import { StyleSheet } from 'react-native';

const styles = StyleSheet.create({
  button: {
    alignSelf: 'flex-start',
    backgroundColor: '#171717',
    borderRadius: 10,
    paddingVertical: 12,
    paddingHorizontal: 20,
  },
  buttonPressed: { opacity: 0.85 },
  buttonDisabled: { opacity: 0.4 },
  text: { color: '#fff', fontWeight: '600' },
});
```

</Tab>

<Tab value="Uniwind">

```tsx
import { clsx } from 'clsx';
import { Text } from 'react-native';
import { Button } from '@limonify/zest-ui';

<Button
  onPress={() => save()}
  className={(state) =>
    clsx('self-start rounded-lg bg-neutral-900 px-5 py-3', {
      'opacity-85': state.pressed,
      'opacity-40': state.disabled,
    })
  }
>
  <Text className="font-semibold text-white">Save</Text>
</Button>
```

## API

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `disabled` | `boolean` | `false` | Ignore user interaction and expose `disabled` on state. |
| `onPress` | `(event) => void` | â€” | Called when the button is pressed. |

Every part also accepts the [common props](/docs/philosophy) â€” `style` (a value or a function of
state), `render`, and `ref`.

### State

| Field | Type | Description |
| --- | --- | --- |
| `pressed` | `boolean` | Whether the button is currently pressed. |
| `disabled` | `boolean` | Whether the button is disabled. |

## State

These fields reach your `style` and `render` functions on **Button** â€” the React Native counterpart of Base UI's `data-*` attributes (`style={(state) => ...}`).
