---
title: Popover
description: A non-modal overlay anchored to a trigger.
---

`Popover` floats content next to a trigger using `@floating-ui/react-native` for collision-aware placement. `Popover.Arrow` points at the anchor.

<Demo>Opening a popover above and below its trigger.</Demo>

## Usage

The behaviour is headless — the **Code** tab is the component; the **StyleSheet** and **Uniwind** tabs are two ways to design it.

<Tabs items={['Code', 'StyleSheet', 'Uniwind']}>

<Tab value="Code">

```tsx
import { Text } from 'react-native';
import { Popover } from '@limonify/zest';

<Popover.Root>
  <Popover.Trigger style={styles.button}><Text style={styles.buttonText}>Settings</Text></Popover.Trigger>
  <Popover.Portal>
    <Popover.Backdrop style={styles.transparent} />
    <Popover.Positioner side="bottom" align="center" sideOffset={8}>
      <Popover.Popup style={styles.popup}>
        <Popover.Arrow style={styles.arrow} />
        <Popover.Title style={styles.title}>Settings</Popover.Title>
      </Popover.Popup>
    </Popover.Positioner>
  </Popover.Portal>
</Popover.Root>
```

</Tab>

<Tab value="StyleSheet">

```tsx
import { StyleSheet } from 'react-native';

const styles = StyleSheet.create({
  button: { alignSelf: 'flex-start', backgroundColor: '#171717', borderRadius: 10, paddingVertical: 10, paddingHorizontal: 18 },
  buttonText: { color: '#fff', fontWeight: '600' },
  transparent: { backgroundColor: 'transparent' },
  popup: { backgroundColor: '#fff', borderRadius: 12, padding: 14, minWidth: 180, borderWidth: StyleSheet.hairlineWidth, borderColor: '#e5e5e5' },
  arrow: { width: 12, height: 12, backgroundColor: '#fff', transform: [{ rotate: '45deg' }] },
  title: { fontWeight: '600' },
});
```

</Tab>

## Accessibility

The trigger carries `aria-haspopup`; the popup is labelled by `Popover.Title` and described by `Popover.Description`. Use a `handle` to open it from a detached trigger.
