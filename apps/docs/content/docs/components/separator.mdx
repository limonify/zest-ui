`Separator` draws a divider between content, with the right accessibility role and orientation.

<Demo>A horizontal and a vertical separator.</Demo>

## Anatomy

<Anatomy code={`Separator`} />

## Usage

The behaviour is headless — the **Code** tab is the component; the **StyleSheet** and **Uniwind** tabs are two ways to design it.

<Tabs items={['Code', 'StyleSheet', 'Uniwind']}>

<Tab value="Code">

```tsx
import { Separator } from '@limonify/zest-ui';

<Separator orientation="horizontal" style={styles.divider} />
```

</Tab>

<Tab value="StyleSheet">

```tsx
import { StyleSheet } from 'react-native';

const styles = StyleSheet.create({
  divider: { height: StyleSheet.hairlineWidth, backgroundColor: '#e5e5e5', marginVertical: 12 },
});
```

</Tab>

<Tab value="Uniwind">

```tsx
import { Separator } from '@limonify/zest-ui';

<Separator orientation="horizontal" className="my-3 h-px bg-neutral-200" />
```

</Tab>

</Tabs>
