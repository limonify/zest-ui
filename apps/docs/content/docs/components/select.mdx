---
title: Select
description: A trigger that opens a list to choose one (or several) values.
---

`Select` shows the chosen label on a trigger and opens an anchored list to pick from. `items` lets `Select.Value` render a label before the popup has ever mounted; set `multiple` for multi-select.

## Usage

The behaviour is headless — the **Code** tab is the component; the **StyleSheet** and **Uniwind** tabs are two ways to design it.

<Tabs items={['Code', 'StyleSheet', 'Uniwind']}>

<Tab value="Code">

```tsx
import { Text } from 'react-native';
import { Select } from '@limonify/zest';

<Select.Root items={FRUITS} value={value} onValueChange={setValue}>
  <Select.Trigger style={styles.trigger}>
    <Select.Value style={styles.text}>{(s) => s.label ?? 'Pick a fruit'}</Select.Value>
    <Select.Icon style={styles.text}><Text>▾</Text></Select.Icon>
  </Select.Trigger>
  <Select.Portal>
    <Select.Backdrop style={styles.transparent} />
    <Select.Positioner side="bottom" sideOffset={4}>
      <Select.Popup style={styles.popup}>
        <Select.List>
          {FRUITS.map((f) => (
            <Select.Item key={f.value} value={f.value} style={styles.item}>
              <Select.ItemText style={styles.text}>{f.label}</Select.ItemText>
              <Select.ItemIndicator><Text>✓</Text></Select.ItemIndicator>
            </Select.Item>
          ))}
        </Select.List>
      </Select.Popup>
    </Select.Positioner>
  </Select.Portal>
</Select.Root>
```

</Tab>

## Animation

`Select.Popup` publishes `state.transitionStatus`. Keep it mounted with `keepMounted` and fade it in and out:

```tsx
import Animated, { FadeIn, FadeOut } from 'react-native-reanimated';
import { Select } from '@limonify/zest';

<Select.Popup keepMounted render={(props) => (
  <Animated.View {...props} entering={FadeIn.duration(120)} exiting={FadeOut.duration(120)} />
)}>
  {/* ... */}
</Select.Popup>
```

See the [Reanimated guide](/docs/guides/animation-reanimated) for the full contract.

## Accessibility

The trigger is `accessibilityRole="combobox"` with `aria-haspopup="listbox"`; items are `role="option"`. In `multiple` mode the popup stays open and `Select.Value` exposes `labels`.
